# NUTS Level 1 Regions GeoJSON

## Purpose
Contains the geographical boundaries of NUTS Level 1 regions (major socio-economic regions). For Belgium, these correspond to the Regions (Flanders, Wallonia, Brussels).

## Context
Used for regional level mapping and analysis.

## Data Shape
**Format**: GeoJSON
**Attributes**: `NUTS_ID`, `NUTS_NAME`, `CNTR_CODE`.

## Lifecycle
Static reference data. Update when NUTS classifications change (every few years).
