# Local Administrative Units (LAU) GeoPackage

## Purpose
Contains the geographical boundaries of Local Administrative Units (municipalities) for Europe/Belgium.

## Context
Used to map data to municipality boundaries. The `4326` in the filename indicates the coordinate reference system (EPSG:4326 - WGS84).

## Data Shape
**Format**: GeoPackage (`.gpkg`)
**Layers**: Typically contains a layer with multipolygon geometries and attributes like `LAU_ID`, `LAU_NAME`.

## Lifecycle
Static reference data. Update when new administrative boundaries are released (usually annually).
