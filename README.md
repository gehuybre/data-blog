# Data Blog

This repository contains the source code and data analyses for the Data Blog.

## Website

The blog is deployed at: **[https://gehuybre.github.io/data-blog/](https://gehuybre.github.io/data-blog/)**

## Project Structure

- `embuild-analyses/`: The Next.js application.
- `embuild-analyses/analyses/`: Contains the MDX posts and data for each analysis.
- `docs/`: Documentation for workflows and files.
- `run_blog.py`: Script to run the local development server.

## Getting Started

1.  Install dependencies:
    ```bash
    cd embuild-analyses
    npm install
    ```

2.  Run the development server:
    ```bash
    python run_blog.py
    ```
